# syst0022-lsd
Github Repository of the Linear System and Design course

## Repository Structure

This repository contains the laboratory notebooks for the SYST0022 course. Each laboratory is organized in its own folder with a corresponding Jupyter notebook.

### Laboratories

- **lab1/** - Laboratory 1 notebook and materials
- **lab2/** - Laboratory 2 notebook and materials
- **lab3/** - Laboratory 3 notebook and materials
- **lab4/** - Laboratory 4 notebook and materials

## Getting Started

To work with the notebooks:

1. Clone this repository
2. Install Jupyter Notebook or JupyterLab
3. Navigate to the desired lab folder
4. Open the `.ipynb` file in Jupyter

## Requirements

- Python 3.x
- Jupyter Notebook or JupyterLab
